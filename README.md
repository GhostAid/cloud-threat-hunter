# CloudTrail Threat Hunter

## Overview
Python tool that analyzes AWS CloudTrail logs and identifies suspicious activities.

## Features
- Detects IAM user creation
- Detects failed access attempts
- Generates detection summaries
- Severity-based alerting

## Technologies Used
- Python
